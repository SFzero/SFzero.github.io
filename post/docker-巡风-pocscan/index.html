Kali下安装docker、巡风、pocscan扫描器

docker
  • 扫描器

近年来，越来越多的公司已经将他们的服务器应用程序从虚拟机（VM）移动到容器。 根据coscale.com的数据，Docker应用的数量在过去的两年中增长了30%， 这个巨大的成功是由于容器比传统的虚拟机需要更少的资源。 为此，在这里，我将学习并记录如何在我喜爱的KALI上安装最流行的容器引擎。

0x01 部署环境

系统：kali2017.2（VM）

0x02 开始部署

  • 1 Docker需要Linux kernels大于3.10并且是64bit的机器，用uname -a可以查看是否符合要求。
  • 2 控制台执行命令：vim /etc/apt/sources.list.d/backports.list加入以下代码，deb http://http.debian.net/debian wheezy-backports main然后刷新源，然后安装一台服务器来管理和下载OpenPGP证书，ca-certificates，并使APT能够通过HTTPS访问源数据和软件包。（直白点就是确认apt添加了https方法和CA cerificates）

接下来，我们添加一个新的PGP密钥。
apt-key adv --keyserver hkp://p80.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D
并且，添加适当的Docker源条目
echo 'deb https://apt.dockerproject.org/repo debian-stretch main' > \ /etc/apt/sources.list.d/docker.list
完成后，重新输入apt-get
apt-get update
此时，安装docker，只需输入：
apt-get install docker-engine
安装完成后，启动Docker服务
service docker start
验证docker是否正常，输入：
docker run hello-world
如果一切顺利，应该看到下图

0x03 docker下安装巡风扫描器

  • 1 创建镜像
    $ docker build -t xunfeng
    或者
    $ docker pull ysrc/xunfeng
  • 2 创建容器
    $ docker run -d -p 8000:80 -v /opt/data:/data ysrc/xunfeng:latest
    把物理机的/opt/data挂载到容器的/data目录下，访问：http://127.0.0.1:8000/正常访问则代表安装成功。
  • 3 镜像信息
  • 4 安装完成后登陆使用

0x04 docker下安装pocscan

  • 1 关于pocscan
    简单介绍一下pocscan吧，Pocscan是一款开源 Poc 调用框架,可轻松调用Pocsuite,Tangscan,Beebeeto,Knowsec老版本POC 按照官方规范编写的 Poc对目标域名进行扫描，通过 Docker 一键部署，麻麻再也不怕搭建麻烦了。
  • 2 环境搭建
    Pocscan源码作者给出的搭建方法，在搭建的时候可能会有遇到一些麻烦。这里测试一下Kali Linux 64 bit（VM）
    安装要求：64位 内核版本3.0.10以上
  • 3 安装流程
  • 4 pocscan使用
    访问一下8090端口。出现登录界面就是搭建成功了。后台和终端的帐号是root,密码是password

    0x05 kali安装chrome浏览器及pocscan插件

    kali安装Chrome请自行百度（一大把），装pocscan插件更容易，直接将代码根目录那个crx文件拖到谷歌浏览器，点一下安装即可。

  • 5 导入POC
    框架搭建好了，就差POC啦，没有POC什么也干不了。
    网络上分享出来的POC地址：https://github.com/Tu-tu-tu/poc
    主要就是bugscan的POC，拿来测试一下还是可以的

到这就全部结束了
